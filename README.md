# base

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

```
npm run lint--fix
```

# 升级依赖包

## 1：安装依赖

```
npm install -g npm-check-updates
```

## 2：检测依赖

```
ncu
```

## 3：更新到最新版本

```
ncu -u
```

### OR

```
cun -a
```
